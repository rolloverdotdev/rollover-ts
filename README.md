# Rollover TypeScript SDK

Type-safe TypeScript SDK for the [Rollover](https://rollover.dev) subscription billing API.

## Packages

- [`@rolloverdotdev/client`](packages/client) Server-side SDK for checking access and tracking usage.
- [`@rolloverdotdev/browser`](packages/browser) Browser SDK with wallet connection, SIWE auth, and x402 payments.

## Docs

Visit [docs.rollover.dev](https://docs.rollover.dev) for guides and API reference.
